# 日経micro + TOPIX 期货策略框架

先确认一下你关注的TOPIX合约——假设是 **TOPIX mini（1000倍）**（和micro配对较合理）；如果是 TOPIX Large（10000倍）仓位匹配逻辑会不同。

## 一、合约规格与配对基础

| 合约 | 乘数 | Tick | 1tick价值 | 名义金额（@当前点位） |
|------|------|------|-----------|----------------------|
| 日経225 micro | 10 | 5円 | 50円 | ~40万円（@40000） |
| 日経225 mini | 100 | 5円 | 500円 | ~400万円 |
| TOPIX mini | 1000 | 0.25 | 250円 | ~280万円（@2800） |
| TOPIX Large | 10000 | 0.5 | 5000円 | ~2800万円 |

**关键点**：日経micro 和 TOPIX mini 名义金额比 ≈ 1:7，做NT spread时 **7手 micro ≈ 1手 TOPIX mini**（当前NT≈14.3水平）。micro的存在让散户/中型账户也能做精细的NT配对，这是它最大的价值。

**交易时段**（2024年11月延长后）：日中 8:45–15:45 / ナイト 16:30–翌6:00。SQ日只有日中。

## 二、核心策略：NT比率价差（NT Spread）

这是你两个合约最自然的组合，也是日本机构最常打的spread。

**信号构造**
```
NT_t = 日経225_t / TOPIX_t
z_t = (NT_t - MA_N) / σ_N     # N 建议 20/60/250日多尺度
```
当前NT处于历史高位区间（成长股/半导体权重推高）。不要纯粹做均值回归——需要叠加**结构性过滤器**：

- `半导体权重因子`：SOX指数/费半 方向（东京电子8035、爱德万6857、信越9735 等占日経权重极大，但TOPIX权重远低）
- `USDJPY`：日元走弱→出口股→日経相对受益（NT↑）
- `美债10Y收益率`：利率↑→价值股跑赢→TOPIX相对强（NT↓）
- `银行股BB信号`：三菱UFJ/三井住友方向代表TOPIX的金融beta

**实战规则（swing级别）**
- |z| > 2 且结构性因子与z同向 → 建仓（反向做NT）
- 头寸按 `名义金额中性` 配（不是按tick value），即上面的 7:1
- 目标：z → 0.5；止损：z 扩大1个σ
- 持仓3–15个交易日

**Python示意**
```python
# N_mic: 日経micro手数, T_mini: TOPIX mini手数
def nt_hedge_ratio(nk, tpx):
    notional_ratio = (nk * 10) / (tpx * 1000)
    # 1手 TOPIX mini 对应多少手 Nikkei micro 保持名义中性
    return round(1 / notional_ratio)  # 当前约7
```

## 三、隔夜缺口与日米联动

日本市场对美股隔夜的响应是**最稳定的alpha来源之一**。

**核心事实**：日経夜盘（16:30–06:00）覆盖美股全时段。日中盘开盘（8:45）的gap基本是夜盘已消化完的行情 → **日中盘gap交易胜率很低**，真正的alpha在夜盘分段反应上。

**可操作分时段**
- `22:30–24:00`（美股开盘时段）：ES/NQ 1分钟动量 → 日経micro 跟随的微观领先滞后约50–200ms，适合做 lead-lag scalping
- `早盘 8:45–9:30`：日中开盘后的 **反转效应**——夜盘到日中切换时流动性断裂常导致过度反应，15分钟均值回归信号较有效
- `14:30–15:15`：机构收盘前对冲，趋势延续性最强

**信号模板**
```python
# 夜盘早段 (16:30-22:00) vs 美股开盘后 (22:30-)
overnight_alpha = nikkei_ret[16:30-22:00] - ES_ret[overlap]
# 若显著偏离，22:30后做收敛
```

## 四、期现基差 & 展期（Calendar）

**理论基差**：`F = S × exp((r - q) × T)`
日本 r≈0.5%（OIS），q（日経年化股息率）≈1.8%，所以**日経期货理论上贴水**。TOPIX股息率更高（~2.3%）贴水更深。

**两个可打的点**：
1. **分红季基差放大**（3月末、9月末）：贴水加深→做多期货同时融券/做空现货篮子（对散户不现实，但可以做 **近月 vs 远月 calendar**，配合分红调整）
2. **SQ周基差收敛**：第二个周五前的基差回归几乎必现，周三-周四介入

**Calendar spread**（同品种近远月）：
- 做空近月 + 做多远月 = 做多 contango（即做空贴水幅度）
- 股息不确定性增大时贴水扩大 → 先卖远月买近月
- 日経micro 可以和 mini 做跨品种calendar，但流动性差，一般还是 mini 做calendar更好

## 五、事件驱动叠加层

- **BOJ会议日**（每年8次）：当日夜盘vol骤升，IV crush机会在结果公布后30分钟内。日経 vs TOPIX 反应差异：鸽派→TOPIX（银行利差收窄）相对弱，NT↑
- **指数年度调整**（日経7月/10月，TOPIX连续）：调整日前后做被纳入/剔除篮子 vs 指数期货的spread
- **メジャーSQ**（3/6/9/12月第二个周五）：SQ价格 vs 前日收盘常有2–3bp的可博弈偏离，周四夜盘末段有固定模式
- **决算季**（4月下旬–5月、7月下旬–8月、10月下旬–11月、1月下旬–2月）：权重股财报日，micro可用来做对冲TOPIX单边的bet

## 七、风险管理要点

1. **单笔风险**：建议 0.3–0.5% 账户 / 笔（NT spread这种低方差策略可放到 0.5–1%）
2. **相关性风险**：所有策略都隐含长日股beta——留一个 `总beta上限`，例如组合整体日経beta不超过 0.3
3. **夜盘gap风险**：持仓过夜需考虑 ES/原油/台海风险事件，建议夜盘仓位 ≤ 日中仓位 50%
4. **SQ结算风险**：SQ日前一周减少近月仓位，避免异常SQ价
5. **资金费率 / 维持保证金**：OSE 期货保证金按 SPAN，波动放大时会被追保——预留30%缓冲

一、核心市场机制理解

产品基础信息
日经225指数期货：以日经225指数为标的的标准化期货合约，是日本最具代表性的股指期货产品。
日经micro（微型日经225指数期货）：合约规模较小的衍生品，以日元计价的合约规模为50日元，以美元计价的合约规模为0.50美元，适合中小投资者参与。
TOPIX指数期货：以东京证券交易所价格指数(TOPIX)为标的的期货合约，反映日本整体股市表现。

价格限制与熔断机制
制限値幅制度：日本交易所采用基于基準値段的百分比限制，例如日経225先物的制限値幅为基準値段的8%（上限）和-8%（下限）。
熔断机制：日本市场没有针对股指的全市熔断，但对期货、期权等衍生品实施交易暂停：
  一、二、三级熔断阈值分别为8%、12%和20%
  熔断触发后，所有月份的期货合约交易暂停10分钟以上
  与美国不同，日本熔断期间不允许任何形式的暗盘交易，这可能导致机构投资者无法进行风险对冲

当前市场动态
日经225指数：2026年4月23日早盘首次突破6万点关口，创下历史新高，但受获利回吐卖压影响，午后回落收低，收盘报59140.23日元。
TOPIX指数：近期走势相对稳健，2026年4月17日最新报价为3,834.73点，较前一交易日上涨0.85%。
市场驱动因素：美股科技股走强、外资持续流入及AI相关产业带动是近期市场情绪升温的主要原因。

二、核心交易策略类型

期现套利策略
基本原理：利用期货市场与现货市场在价格上的差异进行低买高卖获利。
操作方式：
  正向基差套利：当现货指数被低估，期货合约被高估时，卖出期货合约，同时买入成份股。
  反向基差套利：当现货指数被高估，期货合约被低估时，买入期货合约，同时融券卖空成份股。
适用场景：在市场波动较大、基差扩大时尤为有效，例如日经225指数突破6万点后的波动期。

跨市场套利策略
基本原理：同一股指在不同国家的交易所价格出现偏差时，通过"低买高卖"赚取差价。
典型案例：
  日经225指数期货同时在日本大阪、新加坡、芝加哥交易，当大阪报价28000点，芝加哥报价28150点时，可在大阪买入、芝加哥卖出。
  利用汇率波动进行套利，如2023年日元贬值期间，新加坡日经225期货涨幅远超大阪市场。
关键要点：
  市场存在价差：不同市场同一或相关期货品种价格有差异，扣除成本后有盈利空间。
  流动性保障：选择成交量大、持仓量高的期货合约进行操作。
  成本控制：详细核算各市场交易成本，包括手续费、汇率兑换成本等。

技术指标组合策略
核心指标组合：MACD+KDJ+布林带三指标组合策略在历史回测中表现出较高胜率。
策略要点：
  MACD：判断中长期趋势，金叉死叉信号需结合位置判断（零轴上方金叉可靠性更高）。
  KDJ：识别短期超买超卖状态，K值和D值都小于20时进入超卖区间，反弹概率增加。
  布林带：当股价触及布林带上轨时，通常被认为进入超买区域；触及下轨时则进入超卖区域。
动态阈值应用：根据市场状态动态调整指标阈值，例如"趋势初期"RSI>82才触发超买，"震荡市"则55即预警。
重要提醒：单一技术指标策略效果有限，纯MACD策略10年累计亏损率达37.2%，而MACD+KDJ组合在2020-2024年回测中年化收益率达24%，最大回撤控制在18%。

三、风险控制与注意事项

策略风险识别
期现套利风险：
  现货组合的跟踪误差风险
  现货头寸和期货头寸的构建与平仓面临的流动性风险
  追加保证金的风险
跨市场套利风险：
  汇率波动可能抵消利润，例如赚了5%的美元收益，但期间人民币升值3%，实际到手只剩2%
  政策黑天鹅事件，如交易所限仓政策变化
技术指标风险：
  指标本质是滞后滤波器，无法内生表达市场状态切换
  同一数值在不同市场环境下含义截然不同，需结合结构位置、量价配合

风险控制措施
仓位管理：根据市场波动性动态调整仓位，高波动市场中降低仓位和交易频率。
止损纪律：严格执行止损，单笔损失不超过总资金的2%。
多周期验证：要求周线MACD金叉+日线RSI突破动态阈值+60分钟布林带开口扩大同步发生。
量价验证：所有信号必须满足"当日成交额＞20日均值×1.3"且"主力净流入占比＞15%"。

四、具体交易策略建议

日经micro期货交易策略
短线交易策略：
  利用日经micro合约规模小的特点，适合日内短线交易
  结合30分钟和1小时周期的技术指标，当MACD在零轴上方形成金叉且KDJ从超卖区回升时入场
  设置1.5%的固定止损，目标收益率3-5%
波段交易策略：
  关注日经225指数突破关键整数关口（如6万点）后的走势
  采用"突破-回踩"策略，当价格突破前高后回踩确认支撑位时入场
  结合布林带指标，当价格从下轨向上突破中轨且布林带开口扩大时视为买入信号

TOPIX指数期货交易策略
趋势跟踪策略：
  使用50日和200日移动平均线判断长期趋势
  当短期均线向上穿越长期均线且TOPIX指数站稳50日均线时，视为买入信号
  设置动态止损，当价格跌破20日均线时平仓
震荡市策略：
  在TOPIX指数处于1500-2500点区间震荡时，采用"高抛低吸"策略
  当RSI低于30时买入，高于70时卖出，结合布林带下轨和上轨作为参考
  注意当前TOPIX指数已接近历史高点，需警惕超买风险

综合套利策略
期现套利机会：
  监控日经225指数现货与期货的基差，当基差超过3%时考虑套利机会
  由于日经225期货通常存在贴水，可采用"买入期货+卖出股票"的反向套利策略
跨市场套利机会：
  关注日经225指数在大阪、新加坡和芝加哥三地市场的价差
  当价差超过0.5%且流动性充足时，进行跨市场套利操作
  特别注意汇率波动对套利收益的影响，建议选择本币计价市场进行操作

## 先看两个品种差异

日经225 micro 期货是以日经225指数为标的的微型合约，最小交易单位更小，适合用来做更细的仓位管理；TOPIX 迷你期货则以 TOPIX 为标的，合约规模是 TOPIX 期货的 1/10，最小变动单位为 0.25 点，并且有月度限月的连续交易结构。 [jpx.co](https://www.jpx.co.jp/english/derivatives/products/domestic/225micro-futures/index.html)
简单说，**日经 micro 更适合抓波动、做战术进出；TOPIX 更适合做市场整体 beta 暴露、对冲和配对**。 [cmegroup](https://www.cmegroup.com/markets/equities/files/trade-japanese-equity-index-futures-fact-card.pdf)

## 交易逻辑

日经225更集中地反映大型权重股和市场风险偏好，因此对美元/日元、美国科技股、风险情绪更敏感；TOPIX 覆盖更广，更像日本股市整体温度计。 [sc.macromicro](https://sc.macromicro.me/blog/tou-zi-ri-ben-gu-shi-you-nei-xie-guan-jian-zhi-biao-shang)
这意味着，当市场由少数权重股驱动时，日经往往强于 TOPIX；当上涨扩散到更广泛板块时，TOPIX 往往更稳、更“健康”。 [moomoo](https://www.moomoo.com/futures/TOPIXMAIN-JP)
所以策略上可以把两者分成三类用法：**方向交易、对冲交易、价差交易**。 [youtube](https://www.youtube.com/watch?v=uELx_qiKVt4)

## 三、多维度交易策略层面

### 3.1 主策略：顺趋势持有 + 逆指値动态护航（中短线，持仓1～5日）

**具体操作框架：**

| 项目 | 日经225微型期货 (MNI) | TOPIX／Mini TOPIX期货 |
|------|----------------------|-------------------------|
| **进场参考** | 当价格回踩至5日/10日均线不破，RSI由高位回归60-65区间，MACD维持金叉状态时建仓 | 参考2120—2150区间作为中期支撑平台，结合低位成交量放大信号建仓 |

**策略优势：**
- 利用期货24小时交易优势，对重大事件做出快速头寸调整。
- 逆指値注文使得在无法实时盯盘时仍能有效控制亏损，避免出现单向暴跌中无法及时退场的情形。

---

## 1. NT Ratio 统计套利策略 (Nikkei-TOPIX Spread)
这是日本市场最经典的配对交易策略。**NT Ratio = 日经 225 指数 / TOPIX 指数**。

* **逻辑原理：**
    * 日经 225 是价格加权（受高价权重股如迅销、东京电子影响大），更具**进攻性/投机性**。
    * TOPIX 是市值加权（涵盖东证所有股票，受金融、制造等权重大），更具**稳健性/代表性**。
    * 当两者走势背离，NT Ratio 偏离均值时，存在回归需求。
* **操作手法：**
    * **做多 Ratio：** 买入 Nikkei Micro，卖出等价值的 TOPIX 先物（通常利用 Micro 的小规模来精确配比）。
    * **触发条件：** 使用 Z-Score 指标观察 NT Ratio 偏离 20 日均线的程度（如大于 $\pm 2$ 个标准差）。
    * **适用场景：** 当半导体板块暴涨（日经强）或银行/汽车板块补涨（TOPIX 强）后的修复。

---

## 2. 基于 VWAP 的日内均值回归/突破
考虑到日经 Micro 的高频波动特性，利用成交量加权平均价（VWAP）作为“价值中轴”非常有效。

* **策略核心：**
    * **早盘 08:45 - 09:15：** 观察开盘区间。若价格站在 VWAP 之上且放量，视为多头强势。
    * **均值回归：** 当价格偏离 VWAP 超过一定比例（如 $0.5\%$）且出现 K 线反转形态，入场做空博取回归中轴的收益。
* **技术参数：** 结合 **ATR（平均真实波幅）** 动态调整止损位。由于 Micro 只有 5 点一跳，手续费占比极低，适合进行多频次的“刮头皮”交易。

---

## 3. 隔夜缺口（Gap）与美股联动策略
日经指数在早盘（08:45）的开盘价极大程度上受到隔夜美股（S&P 500/Nasdaq）和美元兑日元（USD/JPY）汇率的影响。

* **逻辑原理：** 日本市场常有“开盘即巅峰”或“开盘即谷底”的情况。
* **策略模型：**
    1.  **相关性分析：** 计算前一晚 CME Globex 交易的时段内，CME 日经先物与 S&P 500 的相关系数。
    2.  **缺口回补：** 如果早盘开盘缺口（Gap）过大（例如超过 $1.5\%$），且汇率未出现配合移动，倾向于回补缺口。

- **日经与美股联动**：标普500/纳斯达克前夜表现对日经开盘影响显著

## 四、具体策略类型

### 策略A：隔夜跳空策略（适合日经 micro）

### （一）趋势跟踪策略
**适用场景**：明确的上升/下降趋势中

### （二）区间交易策略

  - 区间内高抛低吸，到达边界时反向操作
  - 仓位控制在轻仓，避免突破风险
  - 突破区间时及时止损，转向趋势策略

### （三）NT倍率套利策略
**核心逻辑**：利用日经与TOPIX的价格差回归特性获利
- **计算方式**：NT倍率 = 日经225指数 ÷ TOPIX指数
- **操作方法**
  - 倍率偏离历史均值时入场：倍率过高→卖日经、买TOPIX；倍率过低→买日经、卖TOPIX
  - 等待倍率回归均值后平仓
  - 适合风险偏好较低的投资者，可对冲部分市场风险

### （四）事件驱动策略

  - 事件前减少仓位，避免不确定性
  - 事件后根据市场反应快速决策，顺势操作
  - 波动加大时，缩小止损幅度，控制风险

  或者当日回撤到 **1%–1.5%** 就停

### 五、合约怎么选

这个细节很重要。

* **平时永远做最近月主力**
* **临近到期前 3–5 个交易日**，观察是否切到下一主力
* 因为这些指数期货的最后交易日，都是**合约月第二个星期五之前的那个营业日**，所以你策略里要提前写好换月规则，避免流动性和价差突然变化。([日本交易所集团][1])

### 六、最适合你的落地路径

我建议你不要一上来就做复杂对冲，按这 3 步走：

**第 1 阶段**：
只做 **日经225 micro 单边突破**，先把交易时段、止损、止盈跑顺。

**第 2 阶段**：
把 **mini-TOPIX** 加进来，只做“确认过滤”，不做双腿配对。

**第 3 阶段**：
再做 **日经 / TOPIX 相对强弱策略**，比如：

* 强势扩散：多日经，弱配对空 mini-TOPIX
* 广度修复：多 mini-TOPIX，弱配对空日经

如果只让我给一个最终建议，那就是：

**个人做日本先物，最值得先做的不是 TOPIX 大合约，而是“日经225 micro 做执行、mini-TOPIX 做过滤”的趋势策略。**
这套最容易上手、最容易控制仓位，也最容易后续写成自动化程序。
